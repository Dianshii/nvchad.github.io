## 如果你刚刚接触 Nim/Neovim

我们强烈建议你先学习一下怎么使用 Neovim/Vim。如果你完全是 Vim 新手，那在你了解 Vim mode、默认 editor commands、global、mapping 等基本知识之前，NvChad会用起来很费劲。

以下是强烈推荐的，也是任何新手 Vimmer 一定会学的：

- Vim tutor

## 认识 lua 

- 强烈建议在配置 neovim config 之前学习基本的 lua 知识。
- 这是一种非常简单且小型的语言，如果您是一名程序员，可能只用花 5-10 分钟。

## 学习 Nvim lua

- 您需要知道如何将 vimscript 代码转换为 lua 来进行配置。
- 这里阅读官方[文档](https://neovim.io/doc/user/lua-guide.html#lua-guide)。
