## 前提条件

- [Neovim 0.8.3](https://github.com/neovim/neovim/releases/tag/v0.8.0)。
- 在终端模拟器内设置了 [Nerd Font](https://www.nerdfonts.com/) 字体。
  - 确认你使用的 nerd font 字体不是以 <strong>Mono</strong> 结尾，以防图标过小。
   - <strong>例如：</strong> Iosevka Nerd Font ，而不是 <s>Iosevka Nerd Font Mono</s>
- [Ripgrep](https://github.com/BurntSushi/ripgrep) is required for grep searching with Telescope <strong>(OPTIONAL)</strong>. 
- 如需使用 Telescope grep 搜索，则需安装 [Ripgrep](https://github.com/BurntSushi/ripgrep)<strong>（可选）</strong>。
- GCC 和 Windows 用户必须已安装 [`mingw`](http://mingw-w64.org/downloads) 并配置环境变量。
- 删除已有的 neovim文件夹（参考以下指令）。

## 卸载
```bash
# linux/macos (unix)
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
rm -rf ~/.cache/nvim

# windows
rd -r ~\AppData\Local\nvim
rd -r ~\AppData\Local\nvim-data
```
