## 条件语句

```lua
-- 数值比较
local age = 10

if age > 18 then
  print("over 18") -- 这里不会执行
end

-- elseif 和 else
age = 20

if age > 18 then
  print("over 18")
elseif age == 18 then
  print("18 huh")
else
  print("kiddo")
end

-- 布尔型比较
local isAlive = true

if isAlive then
    print("Be grateful!")
end

-- 字符串比较
local name = "sid"

if name ~= "sid" then
  print("not sid")
end
```

### 组合语句

```lua
local age = 22

if age == 10 and x > 0 then -- 需要都满足
  print("kiddo!")
elseif x == 18 or x > 18 then -- 满足一个或多个
  print("over 18")
end

-- result: over 18
```

**反转值**

```lua

if not isAlive then
  print(" ye ded!")
end
```

## 作用域

变量有不同的作用域。一旦出了作用域，该作用域内的值将无法再访问。

```lua
function foo()
  local n = 10
end

print(n) -- nil，n 在 foo() 外无法访问
```

## 循环

实现循环的不同方法

### While 循环
```lua
local i = 1

while i <= 3 do
   print("hi")
   i = i + 1
end
```

### For 循环
```lua
for i = 1, 3 do
   print("hi")
end
-- 都会打印三遍 hi
```

## 表（table）

- 表（table）可以用来存储复杂数据。
- 表（table）的类型：arrays (lists) 和 dicts (key,value) 

### 数组（array）

- 其内项目可通过 "index" 索引访问。

```lua
local colors = { "red", "green", "blue" }

print(colors[1]) -- red

-- 遍历列表的不同方法
-- #colors 指表（table）的长度，语法为 #tablename

for i = 1, #colors do
  print(colors[i])
end

-- ipairs 
for index, value in ipairs(colors) do
   print(colors[index])
   -- or
   print(value)
end

-- 如果你不需要使用索引或值，则可以将其替换为 _
for _, value in ipairs(colors) do
   print(value)
end
```

### 字典（dictionary）

- These contain keys and values:
- 以下都包含键值对：

```lua 
local info = { 
   name = "sid",
   age = 20,
   isAlive = true
}

-- 都会打印 sid
print(info["name"])
print(info.name)

-- 用 pairs 循环遍历键值对
for key, value in pairs(info) do
   print(key .. " " .. tostring(value))
end

-- 会打印 name sid, age 20 等等···
```

### **嵌套表（Nested tables）**

```lua
-- 嵌套列表（Nested lists）
local data = {
    { "sid", 20 },
    { "tim", 90 },
}

for i = 1, #data do
  print(data[i][1] .. " is " .. data[i][2] .. " years old")
end

-- 嵌套字典（Nested dictionaries）
local data = {
    sid = { age = 20 },
    tim = { age = 90 },
}
```
